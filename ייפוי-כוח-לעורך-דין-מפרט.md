# ייפוי כוח לעורך דין - תבנית לשימוש עורכי דין

## 📋 למי זה מיועד

**תבנית זו מיועדת לעורכי דין** שרוצים להכין ייפוי כוח ללקוח לחתום עליו.

### איך זה עובד:
1. **אתה (עורך הדין)** ממלא את התבנית
2. **הלקוח** חותם על ייפוי הכוח
3. **אתה מקבל** סמכות לייצג את הלקוח

## מתי להשתמש בכל סוג {#מתי-להשתמש}

| סוג ייפוי כוח | מתי להשתמש | דוגמה |
|---------------|------------|--------|
| **כללי** | כשאתה מטפל בכל הענייים המשפטיים של הלקוח | לקוח קבוע שרוצה שתטפל בכל נושא משפטי שיעלה |
| **מיוחד - ירושה** | כשאתה מטפל רק בצו ירושה/קיום | לקוח שוכר אותך רק להוציא צו ירושה |
| **מיוחד - תיק משפטי** | כשאתה מייצג בתיק ספציפי | לקוח שוכר אותך לתיק תביעה אחד |

## 1. תבנית כללית - לכל עניין משפטי {#תבנית-כללית}

**שימוש:** כשהלקוח רוצה שתייצג אותו בכל עניין משפטי שיעלה.

```markdown
# ייפוי כוח כללי לעורך דין

אני הח"מ:     [שם הלקוח המלא]
               ת.ז. [מספר ת.ז.]
               מען: [כתובת מלאה]
               טלפון: [מספר]
               (להלן: "המיופה" או "הלקוח")

מייפה בזה את כוחו של:

               עו"ד [שם עורך הדין - השם שלך]
               מ.ר. [מספר רישיון שלך]
               כתובת משרד: [כתובת המשרד שלך]
               טלפון: [טלפון המשרד]
               אימייל: [אימייל שלך]
               (להלן: "עורך הדין" או "המיופה כוחו")

נערך ונחתם ביום ________ תאריך __/__/____

---

## סמכויות עורך הדין

אני מסמיך את עורך הדין לייצג אותי ולפעול בשמי ובמקומי בכל העניינים 
המשפטיים הבאים:

### 1. ייצוג כללי

עורך הדין רשאי:

✓ לייצג אותי בפני כל רשות, גוף, מוסד, בנק או אדם
✓ לחתום על מסמכים משפטיים בשמי
✓ לקבל מסמכים, הודעות והחלטות בשמי
✓ להגיש בקשות, תביעות ובקשות לכל גורם
✓ להתגונן מפני תביעות שהוגשו נגדי
✓ להופיע בפגישות ודיונים מטעמי

### 2. ייצוג בבתי משפט ובתי דין

עורך הדין רשאי:

✓ לייצג אותי בכל בתי המשפט ובתי הדין בישראל
✓ להגיש תביעות בשמי
✓ להתגונן בפני תביעות שהוגשו נגדי
✓ להגיש כתבי טענות (כתב תביעה, כתב הגנה, תשובה)
✓ להגיש בקשות ולהשיב על בקשות
✓ להופיע בכל הדיונים
✓ לחקור עדים ולהיחקר
✓ להגיש ראיות
✓ לטעון בעל פה ובכתב
✓ לערער על החלטות וחלופות
✓ לבקש דחיית מועדים

### 3. הליכים מיוחדים

עורך הדין רשאי:

✓ להגיש בקשות לרשם (רשם חברות, רשם מקרקעין, רשם ירושה)
✓ לייצג אותי בהליכי בוררות
✓ לייצג אותי בהליכי גישור
✓ לייצג בפני רשויות מנהליות (משטרה, מס הכנסה, ביטוח לאומי)
✓ לייצג בפני ועדות ערר
✓ להגיש תלונות במשטרה

### 4. הסכמים ופשרות

עורך הדין רשאי:

✓ לנהל משא ומתן בשמי
✓ לחתום על הסכמי פשרה
   ⚠️ **הגבלה:** פשרה בסכום העולה על _______ ₪ דורשת אישורי בכתב
✓ להסכים לתנאים משפטיים
✓ לוותר על זכויות תביעה
   ⚠️ **הגבלה:** ויתור על זכויות דורש אישורי בכתב

### 5. פעולות כספיות

עורך הדין רשאי:

✓ לשלם אגרות משפטיות בשמי
✓ לשלם שכר טרחה לעצמו ולמומחים מטעמי
✓ לקבל כספים המגיעים לי כתוצאה מהליכים משפטיים
   ⚠️ **הגבלה:** סכומים מעל _______ ₪ יועברו ישירות לחשבוני

### 6. סמכויות נוספות

עורך הדין רשאי:

✓ למנות עורכי דין נוספים מטעמי (ייפוי כוח משנה)
✓ לבטל ייפויי כוח קודמים שניתנו על ידי לעורכי דין אחרים
✓ להסב סמכויות לעורכי דין אחרים במשרדו
```

```markdown
## סמכויות עורך הדין - הליכי ירושה בלבד

עורך הדין מורשה לבצע **רק** את הפעולות הבאות:

### 1. הגשת בקשה

✓ להכין ולמלא את הבקשה לצו ירושה / צו קיום צוואה
✓ לחתום על הבקשה בשמי
✓ להגיש את הבקשה לרשם לענייני ירושה
✓ לצרף את כל המסמכים הנדרשים
✓ לשלם את האגרה הנדרשת

### 2. ייצוג בפני הרשם

✓ להופיע בפני הרשם לענייני ירושה בשמי
✓ להשיב לשאלות ובירורים של הרשם
✓ להגיש מסמכים נוספים לפי דרישת הרשם
✓ לתקן טעויות סופר או פרטים בבקשה
✓ לקבל הודעות והחלטות מהרשם

### 3. טיפול בהתנגדויות (אם יוגשו)

✓ לקבל הודעות על התנגדויות לבקשה
✓ לייצג אותי בהליך ההתנגדות
✓ להגיש תגובה להתנגדות
✓ להופיע בדיונים בפני הרשם
✓ לנהל משא ומתן עם המתנגדים
✓ לחתום על הסכם פשרה
   ⚠️ **הגבלה:** פשרה הגורעת מזכויותיי בעיזבון דורשת אישורי בכתב

### 4. קבלת צו הירושה / צו קיום הצוואה

✓ לקבל את צו הירושה / צו קיום הצוואה בשמי
✓ לקבל עותקים מאושרים של הצו
✓ להגיש בקשות לתיקון טעויות בצו

### 5. הוצאות ותשלומים

✓ לשלם אגרות לרשם מטעמי
✓ לשלם עבור העתקים ואישורים
✓ לקבל החזר הוצאות שהוצאו מטעמי
```

## 4. שדות נדרשים למערכת {#types}

```typescript
// src/types/attorney-power-of-attorney.ts

export type AttorneyPOAType = 
  | 'general'           // כללי לכל עניין משפטי
  | 'inheritance'       // מיוחד לירושה
  | 'court_case'        // מיוחד לתיק בית משפט

export interface AttorneyPowerOfAttorney {
  // פרטי הלקוח (המיופה)
  client: {
    name: string
    id: string            // ת.ז.
    address: string
    phone?: string
    email?: string
  }
  
  // פרטי עורך הדין (המיופה כוחו)
  attorney: {
    name: string
    licenseNumber: string  // מספר רישיון
    address: string
    phone: string
    email: string
  }
  
  // סוג ייפוי הכוח
  type: AttorneyPOAType
  
  // עניין ספציפי (לייפוי כוח מיוחד)
  specificMatter?: {
    // לירושה
    inheritance?: {
      deceasedName: string
      deceasedId: string
      dateOfDeath: string
    }
    
    // לתיק משפט
    courtCase?: {
      caseNumber?: string
```
