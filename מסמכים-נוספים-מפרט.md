## 5. זכויות הלקוח

5.1 הלקוח זכאי לבקש מהרשם לענייני ירושה או מבית המשפט לקבוע שכר 
    טרחה שונה מהמוסכם, אם יראה שהסכום אינו סביר.

5.2 הלקוח זכאי לעיין בתיק בכל עת סבירה.

5.3 הלקוח זכאי לקבל עדכונים שוטפים על התקדמות ההליך.

5.4 הלקוח זכאי להפסיק את ההליך בכל עת, ובמקרה כזה ישלם שכר טרחה 
    יחסי בגין העבודה שנעשתה בפועל.

## 6. חובות עורך הדין

6.1 עורך הדין מתחייב:
    - לפעול בשקידה ובמיומנות
    - לעדכן את הלקוח על התקדמות ההליך
    - לייצג את האינטרסים של הלקוח
    - לשמור על סודיות מקצועית

6.2 עורך הדין לא יגבה תשלום נוסף עבור שירותים המפורטים בסעיף 2.2 
    לעיל.

## 7. ביטול ההסכם

7.1 הלקוח רשאי לבטל הסכם זה בכל עת.

7.2 במקרה של ביטול, הלקוח ישלם שכר טרחה יחסי בגין העבודה שנעשתה 
    בפועל עד למועד הביטול.

7.3 עורך הדין ימסור ללקוח את כל המסמכים והתיק תוך 7 ימים מהביטול.

## 8. הוראות כלליות

8.1 הסכם זה מבטל כל הסכם קודם, בכתב או בעל פה, בין הצדדים.

8.3 הודעות ייעשו לכתובות המפורטות בכותרת ההסכם.

## 3. בקשה למינוי אפוטרופוס {#בקשה-למינוי-אפוטרופוס}

### רקע משפטי

**חוק הכשרות המשפטית והאפוטרופסות, התשכ"ב-1962:**

**סעיף 33 - מינוי אפוטרופוס:**
> "בית משפט רשאי למנות אפוטרופוס לחסוי, אם ראה שיש צורך בכך."

**סעיף 34 - שיקולי בית המשפט:**
- טובת החסוי
- רצון החסוי (אם ניתן לברר)
- מערכת היחסים בין החסוי למועמד
- כישוריו של המועמד

**סעיף 35 - חובות האפוטרופוס:**
- לפעול לטובת החסוי
- לנהל את רכושו
- לדאוג לצרכיו
- להגיש דוחות תקופתיים

**תקנות הכשרות המשפטית והאפוטרופסות (סדרי דין), התשכ"ד-1964**

### תבנית בקשה למינוי אפוטרופוס

### עדכון Prisma Schema

```prisma
// prisma/schema.prisma

model Document {
  id         String   @id @default(cuid())
  userId     String
  user       User     @relation(fields: [userId], references: [id])
  
  type       DocumentType
  status     DocumentStatus @default(DRAFT)
  
  // נתוני המסמך (JSON)
  data       Json
  
  // מסמכים מצורפים
  attachments Attachment[]
  
  // חתימות
  signatures Signature[]
  
  createdAt  DateTime @default(now())
  updatedAt  DateTime @updatedAt
  
  @@index([userId])
  @@index([type])
  @@index([status])
}

enum DocumentType {
  TZO_YERUSHA
  TZO_KIYUM
  ATTORNEY_FEE_AGREEMENT
  POWER_OF_ATTORNEY_GENERAL
  POWER_OF_ATTORNEY_SPECIFIC
  GUARDIANSHIP_APPLICATION
}

enum DocumentStatus {
  DRAFT
  IN_PROGRESS
  READY_FOR_SIGNATURE
  SIGNED
  SUBMITTED
  APPROVED
  REJECTED
}

model Attachment {
  id         String   @id @default(cuid())
  documentId String
  document   Document @relation(fields: [documentId], references: [id])
  
  fileName   String
  fileUrl    String
  fileSize   Int
  mimeType   String
  
  uploadedAt DateTime @default(now())
  
  @@index([documentId])
}

model Signature {
  id         String   @id @default(cuid())
  documentId String
  document   Document @relation(fields: [documentId], references: [id])
  
  signerName String
  signerId   String
  signerType String // 'client', 'attorney', 'witness', 'notary'
  
  signedAt   DateTime @default(now())
  ipAddress  String?
  
  @@index([documentId])
}
```

## סיכום

עכשיו יש לך מפרט מלא ל-3 מסמכים נוספים:

1. ✅ **הסכם שכר טרחה** - עם חישוב אוטומטי לפי תקנות
2. ✅ **ייפוי כוח** - כללי ומיוחד (כולל לצו ירושה)
3. ✅ **בקשה למינוי אפוטרופוס** - טופס מלא לבית משפט

כל מסמך כולל:
- רקע משפטי מלא
- תבנית מסמך שלמה
- שדות נדרשים (TypeScript types)
- חוקי תיקוף
- אינטגרציה עם המערכת

המסמכים האלה משלימים את הפלטפורמה ומאפשרים ללקוחות לנהל את כל התהליך המשפטי במקום אחד!
